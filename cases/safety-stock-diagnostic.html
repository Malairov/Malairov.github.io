Safety-Stock Diagnostic

The issue was not one bad formula. New SKUs disappeared from reorder review because several weak control layers interacted: formula behavior, extraction logic, classification logic, filters, workbook governance, and active macro version control.

Symptom: new items vanished from review without obvious errors.

New SKUs with no usage history were being treated as inactive or hidden. The visible symptom looked like a replenishment issue. The real issue was a multi-layer control failure.

FormulaNo usage history can collapse calculated safety stock.Near-zero output created false confidence.
MacroZero usage was classified as DEAD.No explicit no-history state existed.
WorkbookActive logic was not always the edited logic.A workbook-embedded macro version kept old behavior alive.

Fix pattern

LogicRebuild fragile logic in Power Query.Less vulnerable to overwritten worksheet formulas.
ClassificationAdd NO-USAGE state.New/no-history items require exception review, not dead-demand routing.
ResilienceUse dynamic header lookup.Avoid silent filter failure from column shifts.
DebuggingAdd diagnostic columns.Make intermediate logic visible during troubleshooting.
