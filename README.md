# Pavlo Malairov — Technical Operations Portfolio (light editorial redesign)

PMP-certified technical operations and implementation portfolio, presented as a
Vibia-inspired editorial collection catalogue: a light warm canvas, dark editorial
typography, spacious collection-style tiles, and quiet premium motion.

## Structure
- `index.html` — homepage
- `cases/` — three case detail pages (four-operating-loops, smartc-ai-delivery, safety-stock-diagnostic)
- `css/base.css` — homepage design system (light editorial)
- `css/case-pages.css` — case-page design system (light editorial)
- `js/data.js`, `js/app.js`, `js/case-pages.js` — content + interaction
- `assets/Pavlo_Malairov_Resume.pdf` — resume (no phone number)

## Design system
Light warm canvas (`#F3F0EA`), dark graphite typography (`#141414`), warm accent (`#B88A4A`).
Dark surfaces are used only as occasional contrast panels (hero feature panel, contact CTA,
the signal/control/execution scan card), never as the page background. Cards are presented as
calm collection tiles; metrics as refined product-spec labels; navigation is minimal and editorial.

## Deploy (GitHub Pages)
Put all files in the repo root, keep the folder structure, set Pages source to `main` / root.
